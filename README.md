### Installation

Use [Bazaar](https://discuss.flarum.org/d/5151-flagrow-bazaar-the-extension-marketplace) or install manually with composer:

```sh
composer require fof/follow-tags
```

### Updating

```sh
composer update fof/follow-tags
```
